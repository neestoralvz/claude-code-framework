## Command Syntax

### Basic Pattern
```bash
/command [target] [parameters]
```

### Common Parameters
- `--scope [minimal|comprehensive]` - Analysis depth
- `--output [summary|detailed|json]` - Result format
- `--priority [HIGH|MEDIUM|LOW]` - Importance level
